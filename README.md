## Description
Le but est de créer un jeu du morpion opérationnel avec une IA qui puisse jouer contre un humain, avec différents degrés de prédiction des coups/victoires possibles. 

## Installation
Pas de librairies ou package à installer (à part python évidemment)

## Utilisation
Pour jouer au différents modes de jeux, rendez vous dans le fichier "main.py" et assurez vous que le mode de jeu que vous voulez tester ne soit pas en commentaire et que les deux autres le soient !
